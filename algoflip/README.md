## 📖 About

**AlgoFlip** is an interactive web application designed to help developers master Data Structures and Algorithms (DSA) through a fun and engaging random topic generator. Perfect for daily practice, interview preparation, or learning new concepts!

## ✨ Features

- 🎲 **Random Topic Generator** - Get a random DSA topic with a single click
- 🎨 **Beautiful UI** - Modern glassmorphism design with smooth animations
- 📊 **Difficulty Levels** - Topics categorized as Easy, Medium, and Hard
- 🌙 **Dark Theme** - Eye-friendly dark gradient background with dot pattern
- 📱 **Responsive Design** - Works seamlessly on all devices
- ⚡ **Fast & Lightweight** - Built with Vite for optimal performance
- 🔄 **Flip Animation** - Smooth card flip effect on topic change

## 🎯 Topics Covered

The app includes **16 essential DSA topics**:

- Array & String
- Linked List
- Dynamic Programming
- Trees & Graphs
- Recursion
- Sorting & Searching
- Queue & Stack
- Hashing
- Math
- Bit Manipulation
- Sliding Window
- Two Pointers
- Greedy Algorithms
- Backtracking
- Divide & Conquer
- Matrix

## 🛠️ Tech Stack

- **Frontend Framework:** React 18.3
- **Language:** TypeScript 5.6
- **Build Tool:** Vite 6.0
- **Styling:** CSS-in-JS (Inline Styles)
- **Fonts:** Inter (Google Fonts)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/algoflip.git
cd algoflip
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## 📦 Build for Production

```bash
npm run build
```

The optimized production build will be generated in the `dist` folder.

## 🎨 Design Highlights

- **Glassmorphism Cards** - Frosted glass effect with backdrop blur
- **Gradient Backgrounds** - Smooth radial gradients with depth
- **Dot Pattern Overlay** - Subtle grid pattern for visual interest
- **Color-Coded Difficulty** - Green (Easy), Yellow (Medium), Red (Hard)
- **Smooth Animations** - CSS transitions and keyframe animations
- **Hover Effects** - Interactive button and card hover states
